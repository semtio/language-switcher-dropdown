# Language Switcher Dropdown

Плагин добавляет выпадающий переключатель языков с флагами и ссылками. Все данные настраиваются в админке, без хардкода.

## Установка
1. Скопируйте папку `language-switcher-dropdown` в `wp-content/plugins/`.
2. Активируйте плагин в админке WordPress.

## Настройка
В админке откройте **Переключатель языков** и заполните:
- **Языки** — список языков (Код, Название, Ссылка на флаг, Ссылка на страницу).
- **Селектор контейнера (десктоп)** — CSS‑селектор контейнера для вставки переключателя на десктопе.
- **Мобильный брейкпоинт** — ширина в пикселях для мобильного режима.
- **Фолбэк в body на мобайле** — вставлять в `body`, если контейнер десктопа не найден на мобайле.
- **Фиксировать на мобайле** — использовать `position: fixed` для переключателя на мобилах.
- **Позиция на мобайле** — выбор угла экрана (верхний левый, верхний правый, нижний левый, нижний правый).
- **Отступ от края (px)** — расстояние от выбранного угла в пикселях.
- **Открывать по наведению** — открывать меню при наведении на десктопе.

## Как определяется текущий язык
Скрипт сравнивает `location.pathname` с путями из поля **Ссылка на страницу**. Если путь совпадает (кроме `/`), язык считается активным.

## Безопасность
- Все данные проходят серверную очистку.
- Вывод на фронтенде выполняется через JS‑рендеринг без прямого HTML‑хардкода.

## Файлы плагина
- `language-switcher-dropdown.php` — основной файл плагина.
- `assets/lsls.js` — логика переключателя.
- `assets/lsls.css` — стили.
- `assets/admin.js` — логика таблицы в админке.
- `assets/admin.css` — стили админки.
